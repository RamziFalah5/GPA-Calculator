# GPA Calculator

A simple web-based GPA calculator application.

## Features

- Calculate GPA based on course grades and credits
- Add and remove courses dynamically
- Responsive design

## Getting Started

1. Open `index.html` in your web browser
2. Add your courses with grades and credits
3. Click "Calculate GPA" to see your result

## Project Structure

- `index.html` - Main HTML file
- `css/style.css` - Stylesheet
- `js/script.js` - JavaScript logic
- `assets/` - Static assets (images, fonts, etc.)

## Technologies Used

- HTML5
- CSS3
- JavaScript (ES6+)
